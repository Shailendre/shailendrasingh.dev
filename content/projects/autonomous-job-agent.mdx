### What It Does

Job Agent eliminates the manual grind of job hunting. It monitors your Gmail for job alert emails (Indeed, LinkedIn), extracts each listing URL, then dispatches a Claude-powered Playwright browser agent to navigate to the page and fill the application form — cover letter included. Every application lands in a **Pending Review** queue where you approve, give feedback, or reject before anything is submitted.

### Architecture

**Email Ingest** — Gmail API reads unread job alert emails on a configurable schedule (APScheduler). Each listing URL is extracted and deduplicated against SQLAlchemy/SQLite.

**Browser Agent** — Claude + Playwright (with stealth mode via `patchright`) navigates to each job URL. The agent receives your resume and cover letter template as context and fills form fields with tailored content. CAPTCHAs are handled gracefully — the job is flagged "Failed" rather than silently skipped.

**Cover Letter Agent** — A separate Claude call tailors your uploaded `.docx`/`.txt` template for each specific role and company before the browser agent uses it.

**Review UI** — Next.js + Tailwind frontend. Review pending applications, give feedback that gets fed back into the agent, then approve and bulk-submit or submit individually.

**Logging** — Every application (approved, rejected, failed) is appended to a Google Sheet for a persistent audit trail.

### Key Design Decisions

- **Human-in-the-loop by default** — the agent never submits autonomously. All submissions require explicit approval, making it useful rather than reckless.
- `patchright` over raw Playwright — stealth fingerprinting reduces bot detection on platforms like LinkedIn.
- Langfuse for LLM observability — traces every Claude call so prompt regressions are easy to spot.
